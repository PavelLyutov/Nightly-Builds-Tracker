How to use

Welcome to BuildTrack. This is a simple app which gives you an overview on one or more Jenkins jobs. Here you will find useful information on how to use the app.

The 'Dashboard' page

This page presents the last (N)* builds for all (configured jobs)**. The information contained for each job/build is:

- Status (shown as icon (//), which is clickable and will link to the build in question);
- Build number (ID);
- Build description;
- Date/time on which the build was triggered;
- Parameters (Hover on the icon to see the list of parameters);
- Link to the Allure report (if present)*** Click the icon to be taken to the report;
- Link to the full console of the build. Click the icon to be taken to the console log;

*Number of builds shown is configurable through the 'Settings' page;
**Shown Jenkins jobs are also configurable through the 'Settings';
***It is possible that your job does not have an Allure report. The link will still open, but it will most likely show a 404 Not found.


The 'Settings' page

Here you can configure for which Jenkins jobs you want to see builds. Also, you can change the number of last builds shown for each job.
